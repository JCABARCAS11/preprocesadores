Definición

Un preprocesador de CSS es una herramienta que nos permite escribir pseudo-código CSS que luego será convertido a CSS real. Ese pseudo-código se conforma de variables, condiciones, bucles o funciones. Podríamos decir que tenemos un lenguaje de programación que genera CSS.

Sass es una excelente herramienta escrita en Ruby que nos permite crear hojas de estilos estructuradas de scss a css. Una de las principales características de los preprocesadores es que nos permiten tener variables. Gracias a las variables podemos almacenar valores y reutilizarlos en cualquier parte del código. Nos ahorrarán mucho trabajo cuando tengamos que editar un valor que se repite a lo largo de nuestro código.

Requisitos

1) Ruby: Es necesario tener instalado ruby en nuestro sistema: sass es un gem (joya) de ruby. Rubygems es el gestor de paquetes de ruby que provee un formato estándar para distibuir paquetes/librerías (gems). Fue diseñado para manejar fácilmente la construcción, instalación y distribución de librerías.

2) Sublime text.

INSTALACIÓN

1) INSTALACIÓN DE RUBY

1) Accedemos al navegador
2) Copiamos en el buscador el siguiente link Ruby.org
3) Descargamos la version mas reciente y esperamos a que descargue.
4) Ejecutamos el instalador y aceptamos los terminos y condiciones y esperamos a que termine la instalación.
5) Finalizamos la instalacion y ejecutamos la consola.


2) Instalación de sass: Mediante líneas de comandos en cmd


Luego de descargar Ruby e instalarlo, debemos abrir la consola. Para ello podemos o bien entrar a la consola clásica de windows (cmd) o abrir la consola con Ruby, que nos aparecerá en nuestra carpeta de programas.
* En Windows ejecutamos el símbolo del sistema cmd.
Con la consola abierta debemos poner el siguiente comando para instalar SASS
* Gem install sass

Este comando instalara sass y todos los componentes requeridos. Para comprobar que la instalación fu exitosa, copiamos y pegamos el siguiente comando:
* sass –v
Nos imprimirá un mensaje, la instalación fue exitosa.
